# Redirects

## Internal Redirects

For navigation within your Inertia app:

```typescript
app.post('/users', ({ inertia }) => {
  await createUser(body);
  return inertia.redirect('/users');
});
```

This performs an Inertia visit to the new URL without a full page reload.

## External Redirects

For navigation outside your Inertia app:

```typescript
app.get('/external', ({ inertia }) => {
  return inertia.location('https://example.com');
});
```

This performs a full page redirect.

## Redirect with Flash

```typescript
app.post('/contact', ({ inertia }) => {
  await sendMessage(body);
  inertia.flash('success', 'Message sent!');
  return inertia.redirect('/');
});
```

## Redirect After Validation Error

```typescript
app.post('/users', ({ inertia, body }) => {
  const errors = validate(body);
  
  if (errors) {
    inertia.errors(errors);
    return inertia.redirect('/users/create');
  }
  
  // ... create user
});
```

## 303 Redirects

POST/PUT/PATCH/DELETE requests are automatically redirected with a 303 status code:

```typescript
// This will redirect with 303
app.post('/users', ({ inertia }) => {
  return inertia.redirect('/users'); // 303 See Other
});

// GET requests use 302
app.get('/old-page', ({ inertia }) => {
  return inertia.redirect('/new-page'); // 302 Found
});
```
