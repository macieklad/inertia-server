# Docs

Mintlify-hosted documentation for inertia-server.

## Run Locally

```bash
bun run dev
```

Or from the root:

```bash
bun run dev:docs
```

Docs will be available at http://localhost:3001

## Build

Builds are handled automatically by Mintlify's platform on push.

## Editing

Edit `.mdx` files in this directory. Changes are live-updated in dev mode.
